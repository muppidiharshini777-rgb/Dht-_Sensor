# 🌡️ Smart Environmental Monitor using DHT22

## 📌 Project Overview

The Smart Environmental Monitor is an embedded systems project that continuously measures **temperature** and **humidity** using the **DHT22 sensor**. The collected data is displayed in real time and can be used for environmental monitoring, smart homes, agriculture, laboratories, and industrial applications.

This project demonstrates sensor interfacing, embedded programming, and real-time data acquisition.

---

## 🚀 Features

- Real-time Temperature Monitoring
- Real-time Humidity Monitoring
- Continuous Sensor Data Acquisition
- Low Power Operation
- Automatic Data Refresh
- Easy to Expand for IoT Applications

---

## 🛠 Hardware Requirements

- DHT22 Temperature & Humidity Sensor
- ESP32 / Arduino Uno / NodeMCU
- Breadboard
- Jumper Wires
- USB Cable

---

## 💻 Software Requirements

- Arduino IDE
- DHT Sensor Library
- Adafruit Unified Sensor Library

## ▶️ How to Run

1. Clone this repository.
2. Open the project in Arduino IDE.
3. Install the required libraries.
4. Select your board and COM port.
5. Upload the code.
6. Open the Serial Monitor to view live temperature and humidity readings.
